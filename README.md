# 豆瓣快搜

一个极简的 Chrome 浏览器扩展，用于快速跳转到豆瓣搜索。

## 功能

选中文本后，右键菜单选择「豆瓣快搜」，即可快速跳转到豆瓣搜索页面。

## 安装

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目的文件夹

## 使用方式

1. 在任意网页上选中想要搜索的文本
2. 右键点击，选择「豆瓣快搜：%s」
3. 自动在新标签页打开豆瓣搜索结果

## 自定义搜索 URL

插件设置页面支持修改搜索 URL，你可以替换为任意搜索引擎：

1. 右键点击扩展图标，选择「选项」
2. 修改搜索 URL（默认：`https://www.douban.com/search?q=`）
3. 点击「保存」

## 隐私与安全

- **无数据收集**：本扩展不收集任何用户数据
- **无外部请求**：扩展本身不发起任何网络请求
- **本地存储**：使用 `chrome.storage.sync` 本地存储用户设置
- **权限最小化**：仅申请 `contextMenus` 和 `storage` 权限

## 技术栈

- Manifest V3
- Chrome Extension API
